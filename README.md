# Описание проекта
Сайт-проект, созданный с целью показать как использовать изученые навыки на практике. В этом проекте используются технологии:

 HTML, CSS;
 шрифты 'Inter', Arial, sans-serif'
 БЭМ;
 Flex-Grid;
 Медиа-запросы (@media);
 java-script.
